What's about event / CloudNative Days Fukuoka 2019

JapanContainerDaysはコンテナやクラウドネイティブをテーマとした開発者のためのベンダーニュートラルなテックカンファレンスとして誕生し、CNCFやグローバル企業からのスポンサーを受けて実施しました。2019年、ContainerDaysはCloudNative Daysに名称を変え「CloudNative Days Fukuoka 2019」としてここ福岡からスタートします！

JapanConotainerDays (JKD) is a vendor neutral Tech conference for containers and cloud native technology in Japan, CNCF & global top vendors sponsored our event. Container Days will be renamed to CloudNative Days and will start [CloudNative Days Fukuoka 2019] from Fukuoka!!
